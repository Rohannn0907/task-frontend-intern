# TaskFlow- Scalable Dashboardwith Authentication
TaskFlow is a modern, scalable web application built using Next.js and Tailwind CSS, featuring authentication, protected routes, and a responsive dashboard with task management capabilities.
The project focuses on clean frontend architecture, real-world UI patterns, and production-ready scalability.  

## Key Features

### Authentication & Authorization
User login and signup flow
Secure authentication using token-based approach
Protected dashboard routes (accessible only after login)
Logout functionality with proper session handling

### Dashboard
Create new tasks
View task list
Update task details
Delete tasks
Search and filter tasks for better usability

### UI & UX
Create new tasks
View task list
Update task details
Delete tasks
Search and filter tasks for better usability

### Security & Architecture
Route protection implemented using middleware
Token-based authentication flow
Clear separation of concerns
Modular and scalable folder structure
Backend-ready architecture for future API integration

### Tech Stack
Framework: Next.js (React)
Styling: Tailwind CSS
UI Components: shadcn/ui
Icons: Lucide React
Theming: next-themes
Routing & Middleware: Next.js App Router
State Management: React Hooks

### Authentication Flow
User signs up or logs in
Authentication token is issued and stored securely
Protected routes are enforced via middleware
User gains access to the dashboard
Logout clears authentication state and redirects appropriately

### Theme Support
Default light theme for accessibility
Optional dark mode toggle
User theme preference persists across sessions
No hydration issues on refresh
